# HR Data Analysis
## Project Overview

Developed an interactive **Sales Analytics Dashboard** using Power BI to analyze transactional sales data and generate actionable business insights.
The project demonstrates strong capabilities in:

* Data Cleaning & Transformation
* Data Modeling
* DAX Calculations
* KPI Development
* Business Insight Generation
* Executive Dashboard Design

## Business Objective
To build a dynamic dashboard that enables:
* Monitoring of Sales Performance
* Target vs Achievement tracking
* Regional & Store-level Analysis
* Revenue Trend Monitoring
* Identification of Growth Opportunities

- Derived insights to improve operational efficiency
